## RSS阅读指南

一份帮你快速上手的RSS阅读指南。


### RSS怎么用？

高效获取信息，你需要这份RSS入门指南  
https://sspai.com/post/56391

蚁阅 - 让RSS更好用  
https://github.com/anyant/rssant

**蚁阅小技巧:**

1. 输入任意博客地址，会自动查找订阅源，也支持批量输入
2. 在文章列表页面， **Ctrl + 点击** 可以直接打开文章内容
3. 阅读底下的文章时，说明上面的已看过，会自动标为已读(置灰)
